# Вычислитель отличий

Консольное приложение для сравнения двух файлов.
Сравнивать можно `json` и `yaml` файлы.


```bash
// собрать проект
make run-dist
````

```bash
// запустить линтинги и тесты
make test
```

```bash
// запустить утилиту
./app/build/install/app/bin/app
```
